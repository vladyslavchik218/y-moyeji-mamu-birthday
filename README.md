# 🎂 Birthday Celebration Website (Special Gift Project)

Веб-сайт, який я розробив для своєї мами на її день народження.

## 🔗 Демонстрація
https://vladyslavchik218.github.io/y-moyeji-mamu-birthday/

## 🎯 Мета проєкту
Створити інтерактивний подарунок, який поєднує в собі щирі привітання та святкову атмосферу. Цей проєкт допоміг мені відшліфувати навички роботи з анімаціями та адаптивною версткою.

## 🛠 Технології
* **HTML5:** Семантична структура.
* **CSS3:** Анімації (Keyframes), Flexbox, CSS Variables для кольорової гами.
* **Visual Studio Code:** Основне середовище розробки.

## 🔥 Що було реалізовано
- **Чуйний дизайн:** Сайт ідеально виглядає як на смартфоні, так і на великому екрані.
- **Інтерактивні елементи:** Анімовані кнопки, ефекти появи тексту.
- **Clean Code:** Весь проєкт написаний вручну без використання важких бібліотек для максимальної швидкості завантаження.

## 💡 Чому це важливо для мого портфоліо?
Цей кейс демонструє мою здатність:
1. Працювати з конкретним ТЗ (технічним завданням).
2. Створювати унікальний дизайн під конкретну людину/подію.
3. Увагу до деталей та емоційного відгуку користувача.
